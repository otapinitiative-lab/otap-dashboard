Work Desk — Tutorial
Mind Map Tab
▶ Select — click a node to select it; drag to reposition
○ Node — click anywhere on the canvas to place a new node; double-click any node to rename it
— Edge — click a source node, then click a target node to draw a directed edge between them
✕ Delete — click a node to remove it and all its edges; click near an edge midpoint to delete just that edge
⬡ Pan — click and drag the canvas to pan the view

Double-click empty canvas in Select mode also adds a node. All changes save automatically.
Data Tab
Import CSV — click the Import button and select a .csv or .txt file
Paste — copy cells directly from Excel or Google Sheets and press Ctrl+V anywhere on the Data tab
Drag & Drop — drag a CSV file onto the paste zone
First row is always treated as column headers
Clear data — removes the current dataset; this cannot be undone

Data is stored per-user in your browser and does not sync to other operators.
Charts Tab
X axis — select any column as the horizontal axis (usually labels, dates, or categories)
Y axis — select a numeric column; non-numeric values are skipped automatically
Chart types — Bar, Line, Area, Scatter. Line and Area show trends over a sequence; Bar compares categories; Scatter shows correlations
Render — generates the chart; click again after changing columns to refresh
Export PNG — saves the chart as an image file

Statistics (Min, Max, Mean, Median, Trend) are shown automatically below the chart.
Projects & Saving
Save — saves your current mind map and data to the active project
Projects (⊙) — open the project manager to create, switch between, or delete projects. Up to 5 projects per operator
Per-user storage — projects are stored in your browser under your access key. Other operators on the same device will not see your projects
Clear — clears only the currently active tab (mind map, data, or chart). Does not affect other tabs

Work data is local to your browser. It does not sync to the server or other operators.
